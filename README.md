# Visual AI  
**Training Artificial Intelligence Models Through a Visual Interface**

Visual AI is a web-based system that enables users to train artificial intelligence models
through an intuitive **visual interface**, without relying on pre-built machine learning libraries.

The project focuses not only on *using* machine learning algorithms, but on **understanding how they work internally**.

## ğŸ¯ Project Purpose

The main objectives of Visual AI are:

- To implement machine learning algorithms **from scratch**
- To make the model training process **transparent and visual**
- To allow users to:
  - Upload datasets
  - Identify the problem type
  - Manually configure model parameters
  - Observe the training process step by step

Visual AI is designed as both an **educational tool** and an **experimental model development platform**.
